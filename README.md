# TarkovChargeCalculation
This is a app for calculating the charge for trading of Escape from Tarkov.

## How to build it
1. Install vcpkg
```cmd
vcpkg install qt
```

2. Build with cmake
```cmd
cmake -S . -B build -DCMAKE_TOOLCHAIN_FILE=Your vcpkg path/scripts/buildsystems/vcpkg.cmake
cmake --build build
```
